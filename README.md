# java-racingcar-precourse

# 기능 목록

1. 자동차경주 뷰
   - input, output(차수별, 우승자)

2. 자동차 움직임 뷰
   - 자동차가 움직이면 그걸 출력하는 뷰

3. 자동차경주 컨트롤러
   - 입력값 처리 및 자동차 모델에게 전달 및 생성
   - 자동차 경주 게임 진행

4. 자동차경주 서비스
   - 예외처리 후 모델에게 전달
   - 우승자 판별
   - 1회차 경기 기능

5. 자동차 모델(클래스)
   - 이름, 좌표 관련 입출력
   - 좌표 이동 후 뷰에 통보하는 observer
